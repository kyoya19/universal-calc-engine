# v11.0 Juo production value promotion gate entry

## Status

Entry checkpoint for the Juo production value promotion gate phase.

Current baseline:

- #174 v10.3 Juo extracted value metadata completion merged.
- Merge commit: `cfcff15fcb095dc9daa413598afff33215afb7fb`.

## Purpose

This phase defines the minimum promotion gate metadata required before any extracted Juo value metadata can move toward a production value row.

The phase records promotion eligibility metadata only.

It does not accept any real Beast King / Juo probability or reward value.

## Promotion gate scope

A probability promotion gate row should record:

- promotion gate id
- extracted value metadata id
- unit metadata id
- citation id
- source id
- source type
- value domain
- extraction status
- unit applicability
- citation readiness status
- source conflict status
- numeric readiness status
- normalization readiness status
- review readiness status
- promotion eligibility
- execution eligibility

A reward promotion gate row should record:

- promotion gate id
- extracted value metadata id
- unit metadata id
- citation id
- source id
- source type
- value domain
- extraction status
- unit applicability
- citation readiness status
- source conflict status
- numeric readiness status
- accounting readiness status
- normalization readiness status
- review readiness status
- promotion eligibility
- execution eligibility

## Required separation

Promotion eligibility is not production execution eligibility.

A row may be eligible for later production value review only after citation, unit, extraction, normalization, conflict, and review readiness have been satisfied.

Even then, this phase does not create production values.

## Boundary

This checkpoint is documentation-only.

No real probability values, reward values, production fixture values, expected value assertions, executable production graph, runtime target substitution, report adapter changes, formatter changes, contribution output changes, UI changes, TeX changes, CSS changes, Android native changes, or Sugoroku baseline changes are included.
