# 成果還元関数 external brief

成果還元関数は、観測・仮定・評価済みの情報を分離し、期待値・勝率・時給・判断根拠へ還元する中核評価モデルです。

## Positioning

- 成果還元関数は、万能計算機の評価方針を説明する名称です。
- キヨタンは、既知の入力から結果を評価する順方向の利用方向です。
- セイカタンは、観測結果から条件を推定する逆方向の利用方向です。

## What it keeps separate

- observed inputs
- assumptions
- evaluated results
- decision output

## What it does not change yet

- code-level type names
- function names
- executable production graph
- source-backed numeric values
- UI / TeX / CSS / Android native output

## External message

この段階では、成果還元関数を外向け説明の中心名称として固定します。

実装上のリネームやproduction値投入は、別工程で安全に分離して扱います。
