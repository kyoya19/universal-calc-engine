# v10.0 Juo extracted value metadata entry

## Status

Entry checkpoint for the Juo extracted value metadata phase.

Current baseline:

- #170 v9.3 Juo unit metadata completion merged.
- Merge commit: `51c2b8958904237dc47b981944f5af2867c38674`.

## Purpose

This phase defines the minimum extracted value metadata boundary required before any source-backed Juo candidate can move toward a production value row.

The phase records extraction status and value-shape metadata only.

It does not accept any real Beast King / Juo probability or reward value.

## Extracted value metadata scope

A probability extracted value metadata row should record:

- extracted value metadata id
- unit metadata id
- citation id
- source id
- source type
- value domain
- extraction status
- extracted expression status
- numerator status
- denominator status
- decimal status
- normalization status
- review status
- execution eligibility

A reward extracted value metadata row should record:

- extracted value metadata id
- unit metadata id
- citation id
- source id
- source type
- value domain
- extraction status
- extracted expression status
- payout amount status
- reward unit status
- accounting basis status
- normalization status
- review status
- execution eligibility

## Required separation

Extracted value metadata is not a production value.

A candidate with extracted value metadata may still be:

- non-executable
- unverified
- conflicted
- missing source-backed numeric extraction
- missing normalized probability representation
- missing normalized reward representation
- waiting for conflict review
- waiting for production value promotion gate
- waiting for production graph binding

## Boundary

This checkpoint is documentation-only.

No real probability values, reward values, production fixture values, expected value assertions, executable production graph, runtime target substitution, report adapter changes, formatter changes, contribution output changes, UI changes, TeX changes, CSS changes, Android native changes, or Sugoroku baseline changes are included.
