## Purpose

This phase defines how a non-executable source candidate may move toward review without becoming an executable production value.

The review gate is an intermediate status layer between inventory and value promotion.

## Review gate scope

A candidate may enter review only when the following are recorded:

- source id
- source type
- source title or label
- expected category
- expected unit category
- citation availability
- retrieval or observation status
- transcription status
- confidence status
- conflict status
- review eligibility
- execution eligibility

## Review gate statuses

Allowed review statuses:

- not_started
- needs_source
- needs_citation
- needs_transcription
- needs_unit_mapping
- needs_conflict_review
- ready_for_review
- rejected
- excluded
