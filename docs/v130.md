# v13.0 Juo production value placeholder entry

## Status

Entry checkpoint for the Juo production value placeholder phase.

Current baseline:

- #182 v12.3 Juo production value draft metadata completion merged.
- Merge commit: `226362586bfb7f2ef1895aae53f87d3dc25be0e2`.

## Purpose

This phase defines the minimum non-executable placeholder boundary required before any Juo row can later become a source-backed production value.

The phase records placeholder production value rows only.

It does not accept any real Beast King / Juo probability or reward value.

## Placeholder scope

A probability production value placeholder row should record:

- production value placeholder id
- draft metadata id
- promotion gate id
- extracted value metadata id
- unit metadata id
- citation id
- source id
- source type
- value domain
- placeholder status
- value readiness status
- normalization readiness status
- promotion eligibility
- execution eligibility

A reward production value placeholder row should record:

- production value placeholder id
- draft metadata id
- promotion gate id
- extracted value metadata id
- unit metadata id
- citation id
- source id
- source type
- value domain
- placeholder status
- value readiness status
- accounting readiness status
- normalization readiness status
- promotion eligibility
- execution eligibility

## Required separation

A production value placeholder is not a source-backed production value.

A placeholder row must not contain:

- source-backed numeric values
- numerator values
- denominator values
- decimal values
- payout amounts
- executable probability fields
- executable reward fields
- production graph bindings

## Boundary

This checkpoint is documentation-only.

No real probability values, reward values, production fixture values, expected value assertions, executable production graph, runtime target substitution, report adapter changes, formatter changes, contribution output changes, UI changes, TeX changes, CSS changes, Android native changes, or Sugoroku baseline changes are included.
