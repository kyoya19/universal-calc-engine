## Current guarantee

A Juo source candidate can now pass through source policy, candidate inventory, review gate, citation metadata, and unit metadata layers without becoming executable.

The unit metadata layer records whether a probability or reward unit is applicable, not applicable, or excluded, but it does not contain source-backed values.
