# 成果還元関数

成果還元関数は、観測・仮定・評価済みの情報を分離し、期待値・勝率・時給・判断根拠へ還元する評価モデルです。

旧称の打技還元関数は廃止し、今後は成果還元関数に統一します。

キヨタンは順方向評価、セイカタンは逆方向推定を担います。
