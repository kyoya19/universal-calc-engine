## Required separation

Source-backed production value draft metadata is not a production value.

A draft row may describe the metadata shape needed for later source-backed production value creation, but it must not contain:

- source-backed numeric values
- numerator values
- denominator values
- decimal values
- payout amounts
- executable probability fields
- executable reward fields
- production graph bindings
- runtime target substitutions
- expected value assertions

## Boundary

This checkpoint is documentation-only.

No real probability values, reward values, source-backed production value rows, production fixture values, expected value assertions, executable production graph, runtime target substitution, report adapter changes, formatter changes, contribution output changes, UI changes, TeX changes, CSS changes, Android native changes, or Sugoroku baseline changes are included.
