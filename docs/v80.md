## Purpose

This phase defines the minimum citation metadata shape required before a reviewed candidate can move toward a source-backed row.

The phase does not accept any real Beast King / Juo probability or reward value.

## Citation metadata scope

A citation metadata row should record:

- citation id
- source id
- source type
- source title or label
- source location
- retrieval or observation status
- retrieval or observation date status
- transcription status
- confidence status
- conflict status
- review status
- execution eligibility

## Required separation

Citation metadata is not a production value.

A candidate with citation metadata may still be:

- non-executable
- unverified
- conflicted
- outside scope
- waiting for unit mapping
- waiting for value extraction
