## Purpose

This phase defines a preflight metadata boundary before any source-backed Juo production value row can be created.

The phase is still metadata-only.

It does not insert real Beast King / Juo probability values or reward values.

## Required separation

Source-backed production value preflight metadata is not a production value.

A preflight row may describe whether a readiness review row is eligible for later source-backed production value creation, but it must not contain:

- source-backed numeric values
- numerator values
- denominator values
- decimal values
- payout amounts
- executable probability fields
- executable reward fields
- production graph bindings
- runtime target substitutions
- expected value assertions

## Boundary

This checkpoint is documentation-only.

No real probability values, reward values, production fixture values, expected value assertions, executable production graph, runtime target substitution, report adapter changes, formatter changes, contribution output changes, UI changes, TeX changes, CSS changes, Android native changes, or Sugoroku baseline changes are included.
