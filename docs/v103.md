## Current guarantee

A Juo source candidate can now pass through source policy, candidate inventory, review gate, citation metadata, unit metadata, and extracted value metadata layers without becoming executable.

The extracted value metadata layer records whether a probability or reward extraction is applicable, not applicable, excluded, or not started, but it does not contain source-backed numeric values.
