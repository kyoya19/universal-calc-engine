# v12.0 Juo production value draft metadata entry

## Status

Entry checkpoint for the Juo production value draft metadata phase.

Current baseline:

- #178 v11.3 Juo production value promotion gate completion merged.
- Merge commit: `eacdb0a86427e8b30873a8273768aa615598a362`.

## Purpose

This phase defines the minimum draft metadata boundary required before any Juo row can later become a production value.

The phase records production-value draft shape metadata only.

It does not accept any real Beast King / Juo probability or reward value.

## Draft metadata scope

A probability production value draft metadata row should record:

- draft metadata id
- promotion gate id
- extracted value metadata id
- unit metadata id
- citation id
- source id
- source type
- value domain
- value shape status
- fraction shape status
- decimal shape status
- normalization shape status
- promotion eligibility
- execution eligibility

A reward production value draft metadata row should record:

- draft metadata id
- promotion gate id
- extracted value metadata id
- unit metadata id
- citation id
- source id
- source type
- value domain
- value shape status
- amount shape status
- reward unit shape status
- accounting shape status
- normalization shape status
- promotion eligibility
- execution eligibility

## Required separation

Production value draft metadata is not a production value.

A draft metadata row may describe the eventual shape of a probability or reward value, but it must not contain:

- source-backed numeric values
- numerator values
- denominator values
- decimal values
- payout amounts
- executable probability fields
- executable reward fields

## Boundary

This checkpoint is documentation-only.

No real probability values, reward values, production fixture values, expected value assertions, executable production graph, runtime target substitution, report adapter changes, formatter changes, contribution output changes, UI changes, TeX changes, CSS changes, Android native changes, or Sugoroku baseline changes are included.
