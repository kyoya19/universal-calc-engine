## Required separation

Source-backed production value promotion metadata is not a production value.

A promotion row may describe whether a source-backed production value draft row is still blocked or eligible for later materialization, but it must not contain:

- source-backed numeric values
- numerator values
- denominator values
- decimal values
- payout amounts
- executable probability fields
- executable reward fields
- production graph bindings
- runtime target substitutions
- expected value assertions

## Boundary
